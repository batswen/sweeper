# a minesweeper game

A simple minesweeper game based on ideas shown in this video: https://www.youtube.com/watch?v=rxdGAKRndz8
## download & installation
 1. clone the repo
  `git clone https://github.com/batswen/sweeper.git`

 2. enter directory
  `cd sweeper`

 3. install dependencies
  `npm install`

 4. run sweeper
  `npm start`
